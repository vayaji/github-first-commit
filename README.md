# GitHub First Commit

https://github-first-commit.vercel.app/

GitHub API を使って、GitHub のリポジトリの最初のコミットの URL を取得するウェブアプリです。

## 技術スタック

-   [Preact](https://preactjs.com/)
-   [TailwindCSS](https://tailwindcss.com/)
-   [Vite](https://vite.dev/)
-   [Vercel](https://vercel.com/)
